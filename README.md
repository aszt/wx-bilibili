### API路径

接口名称|接口路径|备注
--|:--:|--:
首页导航|/navList|
首页轮播图|/swiperList|
视频列表|/videosList|

推荐视频|/othersList?id|需要带上视频的id
评论列表|/commentsList?id|需要带上视频的id
